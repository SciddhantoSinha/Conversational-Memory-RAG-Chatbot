# Conversational RAG Chatbot

A Retrieval-Augmented Generation chatbot that remembers past interactions and answers questions using document context and conversational memory.

## Features
- PDF-based RAG
- Conversational memory
- Multi-turn dialogue
- Context retention
- FAISS semantic search
- Streamlit UI

## Tech Stack
Python, LangChain, FAISS, Sentence Transformers, Streamlit

## Run
pip install -r requirements.txt  
streamlit run app.py
